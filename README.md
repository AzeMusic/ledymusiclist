- BOT TOKEN - [@Botfather](https://t.me/BOTFATHER) botundan ala bilərsiniz.
- PLAYLIST NAME - Playlist kanalı yaradın və kullanıcı adı yazın.
- BOT OWNER - kullanıcı adınız
- PLAYLIST ID - Yaratdığınız playlist  kanal ID
